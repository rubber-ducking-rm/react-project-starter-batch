# react-project-starter-batch

#### `create-react-app`の処理を自動化

- --template typescript を利用
- vscode を利用
- eslint を利用
- prettier を利用
- Atomic Design を利用
- GENERATE_SOURCEMAP を無効化

## Dependency

- Windows: 11
- VScode: 1.73.1

## Usage

```bash
reactTypeScript.bat
type project name :'Your Project Name'
```
